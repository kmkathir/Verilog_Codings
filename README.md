# Verilog Coding Practice

This repository contains Verilog HDL programs written for
Digital Electronics and VLSI fundamentals.

## Topics Covered
- Logic Gates
- Half Adder & Full Adder
- Multiplexers (2:1, 4:1, 8:1)
- Adder & Subtractor
- Flipflops

## Tools Used
- Icarus Verilog
- GTKWave
- VS Code
- EDA Playground
